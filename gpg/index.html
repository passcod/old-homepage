GPG Keys and Signatures

2013·13·1 ⚔ Monday

I generate GPG keys that have an expiry date. Hence, new keys are signed using old keys and and old keys are signed using new keys to ensure trust is kept continuous. Keys are named passcodNN, where NN is the number of the key. Keys 00 to 02 were lost in a previous disk disaster.
